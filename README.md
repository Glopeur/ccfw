# CCFW (Crouch Cancel Frost Walker)


#### FR

Nécessite:
  - Minecraft version 1.14 ou plus récent
  - Serveur Bukkit, Spigot ou PaperSpigot version 1.14 ou plus récent
  
Comment utiliser ?
  - Equiper des bottes ayant l'enchantement Frost Walker
  - S'accroupir afin d'annuler l'effet produit par l'enchantement
  
---

#### EN

Need:
  - Minecraft 1.14 or newer
  - Bukkit, Spigot or PaperSpigot powered server 1.14 or newer

How To Use ?
  - Equip Boots with Frost Walker enchantment
  - Sneak to cancel the enchantment effect
